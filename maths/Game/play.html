Solving Systems Using Elimination

Created by: Devan Thiel
Math Topic: Solving Systems of Equations Using Elimination

How To Play

Players take turns solving systems of equations using the elimination method.

All answers are rounded to 2 decimal places.

  • Player 1 answers first, then Player 2.
  • Correct answers give +1 point.
  • Incorrect answers subtract 1 point.

Time Mode: When time runs out, the player with the most points wins.

Point Mode: The first player to reach the point goal wins.

Game Settings









Player 1: 0 | Player 2: 0
Turn: Player 1
